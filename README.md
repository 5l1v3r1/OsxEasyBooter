# OsxEasyBooter
An easy program for OSX in python that uses diskutil suite to make an USB bootable using whetever iso

 ![img](https://i.imgur.com/5f1tjYU.png)

# Requirements

```
brew install python3
```

# Usage

```
python3 easybooter.py
```
